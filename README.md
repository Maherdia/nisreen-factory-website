# Nisreen Factory Website

Official website for Nisreen Factory (Special & Safety Work - SSW), a Saudi manufacturer specializing in engineered security solutions.

## Features

- Modern React + Vite website
- Product catalog
- Project portfolio
- Certification page
- Contact page
- Responsive design
- Product search
- Project filtering by sector

## Technologies

- React
- Vite
- React Router
- CSS3
- React Icons
